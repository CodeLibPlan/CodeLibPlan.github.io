# 1. 분석할 안드로이드 앱

충남대학교에 관련된 어플리케이션으로, 버스에 관한 앱입니다.

# 2. 개발 환경

Android Studio를 이용하여 개발한 어플리케이션이며, 이 포스팅에 추가할 사진들은 github상에서 캡쳐합니다.

# 3. 내용

여러 개의 파일 중 AdminActivity, LocationActivity에서 네트워크를 사용하는 것을 발견하였으며, 관련 기능을 일부 수정하였습니다.
